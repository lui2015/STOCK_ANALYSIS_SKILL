# stock_analysis · 使用说明

## 一句话介绍

输入一只股票的名字或代码，自动生成一份**可视化深度分析 HTML 报告**——包含实时快照、近五年股价中位数、近五年总市值变化、PE Bands估值、四派圆桌讨论、操作路线图。

---

## 如何触发

在 WorkBuddy 对话中，用以下任意方式触发：

| 触发方式 | 示例 |
|---------|------|
| `@skill:stock_analysis` + 股票名 | `@skill:stock_analysis 安克创新` |
| `@skill:stock_analysis` + 股票代码 | `@skill:stock_analysis sz300866` |
| 自然语言（需含分析意图） | "帮我分析比亚迪" / "宁德时代怎么看" |

**不会触发**的场景：纯行情查询（"宁德现在多少钱"）、板块讨论（"新能源怎么样"）、新闻浏览。

---

## 输入与输出

| 项 | 说明 |
|----|------|
| **输入** | 股票名称（如"安克创新"）或代码（如"sz300866"）；可选附加关注维度 |
| **输出** | 一个完整 HTML 文件，自动预览并作为附件交付 |
| **文件名** | `<股票名拼音或英文名>_analysis.html`，保存在当前工作目录 |

## 报告包含的模块

生成的 HTML 报告从上到下共 **9 大模块**：

```
┌─────────────────────────────────┐
│  1. 头部                        │  股票名、代码、实时价格、涨跌幅
├─────────────────────────────────┤
│  2. 实时快照                     │  市值/PE/成交额/换手率/52周高低（卡片网格）
├─────────────────────────────────┤
│  3. 近五年股价中位数              │  年度表格 + Canvas 柱状图 + 今日价标注线
├─────────────────────────────────┤
│  4. 近五年总市值变化              │  年度表格 + Canvas 柱状图（红涨绿跌）
│     └─ 同比变化百分比             │  当前市值参考线 + 5年均值参考线
├─────────────────────────────────┤
│  5. PE Bands 估值带              │  25%/50%/75%分位价格带
├─────────────────────────────────┤
│  6. 宏观环境快扫                  │  上证/沪深300/恒指 + 宏观要闻
├─────────────────────────────────┤
│  7. 圆桌讨论（四派）              │  产业趋势派 / 估值派 / 基本面派 / 信号派
│     ┌─ 分析框架                  │  每派独立卡片，左侧色条区分
│     ├─ 关键数字                  │  蓝=趋势 / 橙=估值 / 绿=基本面 / 紫=信号
│     ├─ 详细观点                  │
│     └─ 态度标签                  │
├─────────────────────────────────┤
│  8. 总结                         │
│     ├─ 四派态度 + 一句话总结      │  每个派别的核心观点浓缩
│     ├─ 给小白的一句话             │
│     ├─ 操作路线图                 │  不同风险偏好 → 不同路径
│     ├─ 最值得验证的预测           │
│     └─ 最值得重视的风险           │
├─────────────────────────────────┤
│  9. 免责声明                     │
└─────────────────────────────────┘
```

## 四派分析框架

| 派别 | 图标 | 核心视角 | 典型输出 |
|------|------|---------|---------|
| 🌊 产业趋势派 | 蓝色 | 产业链拆解，判断驱动力阶段和最受益环节 | 驱动力判断 + 估值硬约束 |
| 🧮 估值派 | 橙色 | PEG / PE Bands 定价，资金面校验 | 估值工具选择 + 分位 + 价格带 |
| 🔬 基本面派 | 绿色 | 财报审查，管理层评估 | 营收利润拆解 + 管理层靠谱度 |
| 📡 信号派 | 紫色 | 四层信号对齐（政策/产业/资讯/资金） | 信号状态 + 落入组合 + 建议动作 |

### 估值工具自动选择

- 盈利增速 **>20%** → PEG（PE ÷ 增速%）
- 盈利增速 **<15%** → PE Bands（5年分位 25%/50%/75%）
- 盈利为负 → PS 或其他替代指标

## 数据来源与铁律

所有数字**均来自实时查询**，不会用模型记忆数据填充：

| 数据 | 来源 |
|------|------|
| 行情 / 财报 / K线 / 资金 / 技术指标 | westock-data skill（`node scripts/index.js`） |
| 宏观新闻 | 财联社 API + WebSearch 备选 |
| 机构评级 / 一致预期 / 股东结构 | westock-data skill |

**数据铁律**：
1. 先查后写——所有数字有对应查询动作
2. PE 交叉验证——quote 取值 vs 股价÷EPS 反算，偏差>5% 标注
3. 时效标注——盘后 2 小时以上必须标"数据截止"
4. 异常主动提——PE 为负、机构覆盖<3家、股东户数暴增等
5. 货币单位——港股港元/美股美元，不混用¥

```

```

### 示例 2：分析港股
```
@skill:stock_analysis 腾讯控股
```
→ 生成 `tencent_analysis.html`（注意货币单位为港元）

```
@skill:stock_analysis 比亚迪，重点关注估值和资金面
```
→ 报告中估值派和信号派的分析会更详尽

## 注意事项

- ⏱ 完整分析约需 2-3 分钟（15+ 次数据查询 + HTML 生成）
- 📊 支持 A 股、港股、美股，自动匹配对应资金查询命令
- 🎨 涨跌颜色遵循中国市场惯例：**涨红跌绿**
- ⚠️ 报告末尾包含免责声明，不构成投资建议
